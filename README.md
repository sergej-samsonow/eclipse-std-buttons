# eclipse-std-buttons
Copy paste undo and redo buttons for eclipse

### Install
Pack ```stdbtn_1.0.0.jar``` to eclipe/plugins directory.

### Create jar
Pack content from stdbtn into zip archive:
```
stdbtn_1.0.0.zip
    META-INF
        MANIFEST.MF
    plugin.xml
```
and rename zip archive to jar.
